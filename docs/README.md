# 미션 - 크리스마스 프로모션 🌲

## 컨셉

서버와 클라이언트가 통신하는 구조. ( Request, Response )

### 📱 APP

- App 은 기본적으로 입력값을 제한 할 수 있다. ex) 숫자형 입력, 날자형 입력
- App은 실행 시 서버에 있는 음식 리스트를 받아온다. ex) 키오스크
